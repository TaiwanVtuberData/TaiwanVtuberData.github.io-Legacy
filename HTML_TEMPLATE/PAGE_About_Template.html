About The Site

Terms Of Service

Last Updated: 2021/12/31

臺灣 Vtuber 列表 ("The site") uses YouTube API Services to keep recorded VTuber YouTube channels data up-to-date.
The data collected are as followed:

  1. Channel subscribers count
  2. Channel subscribers count difference in last 7 days / 30 days
  3. Channel median video view count in last 30 days
  4. Channel highest viewed video in last 30 days

By using this site, you hereby agreed to be bound by YouTube Terms Of Service

The site uses Twitch API to keep recorded VTuber Twitch channels data up-to-date.
The data collected are as followed:

  1. Channel followers count
  2. Channel median past broadcast view count in last 30 days
  3. Channel highest viewed past broadcast in last 30 days
